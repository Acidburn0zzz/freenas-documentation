{{% pageinfo %}}
Pour modifier rapidement la documentation, vous devez disposer d'un [Compte Github](https://github.com)!
{{% /pageinfo %}}

Si vous trouvez quelque chose à améliorer dans un article, cliquez sur **Edit this page** pour afficher le texte de l'article.
Vous pouvez modifier le texte selon vos besoins et utiliser l'onglet **Preview changes** pour voir s'il y a des bogues dans la syntaxe Markdown.

Lorsque vous avez terminé d'apporter et de tester les modifications, faites défiler la page jusqu'à la section **Commit changes** et rédigez un résumé et une description de vos modifications.

Sélectionnez **Create a new branch for this commit and start a pull request.** et cliquez sur **Propose file change**.
Assurez-vous que vous êtes satisfait de votre résumé et de votre description, puis tapez ** x ** dans le ** [] ** pour accuser réception de la licence de documentation et cliquez sur **Create pull request**.

C'est ça! D'autres contributeurs examineront et fusionneront votre contribution!
